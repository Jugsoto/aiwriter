# 神笔写作 v0.9.3 | 首个公开发布版本

## 中文介绍

欢迎来到 **神笔写作** 的首个公开发布版本。

神笔写作是一款面向中文创作者的 AI 桌面写作工具，专注于把“作品管理、正文创作、设定维护、AI 辅助”整合进同一个本地应用中。这个版本标志着项目第一次以可下载、可安装、可升级的形式正式发布。

### 核心亮点

- 三栏式写作工作台：章节列表、正文编辑器、AI 助手同屏协作，减少频繁切换页面的打断感。
- 本地作品管理：支持书籍、章节、全局设定的组织与维护，适合中长篇内容创作。
- AI 写作辅助：支持正文创作、章节扩写、章节梗概、内容评估等常见创作场景。
- 智能设定与记忆：可以围绕角色、世界观、词条等信息建立结构化设定，并通过向量检索增强上下文一致性。
- 创作工具集成：内置脑洞生成器、书名生成器、简介生成器等实用工具，方便从灵感到成稿的完整流程。
- 桌面端体验：基于 Electron 构建，支持本地安装、独立窗口、数据本地存储，更适合作为日常写作工作台。

### 这个版本包含的能力

- 作品与章节管理
- 自动保存与编辑辅助
- 模型与功能配置管理
- 用量统计与数据管理
- 设定记忆 / 章节记忆相关能力
- AI 创作、扩写、摘要与评估工具
- 首次接入 GitHub Releases 自动更新能力，为后续版本升级做好基础设施准备

### 首发说明

- 这是项目的 **第一个正式发布版本**。
- 当前发布面向 **Windows 安装包**。
- 自动更新已接入 GitHub Releases，后续版本将可以在应用内检查、下载并安装更新。

### 适合谁

- 正在写小说、网文、长篇故事的作者
- 需要维护角色设定、世界观、章节结构的创作者
- 希望在本地桌面环境中使用 AI 写作辅助工具的用户

感谢你体验神笔写作的首个公开版本。欢迎继续反馈使用体验与改进建议。

## English Introduction

Welcome to the **first public release** of **AIWriter**.

AIWriter is a desktop writing application designed for fiction and long-form content creation. It brings project management, drafting, worldbuilding, and AI-assisted writing into one local desktop workspace. This release marks the first time the project is available as an installable and updatable application.

### Highlights

- Three-panel writing workspace: manage chapters, write content, and collaborate with the AI assistant in one focused layout.
- Local project management: organize books, chapters, and global settings for long-form writing workflows.
- AI writing assistance: supports drafting, chapter expansion, summaries, and content review.
- Structured memory and settings: maintain character, worldbuilding, and terminology data with vector-based context support.
- Built-in creative tools: includes idea generation, title generation, and synopsis generation to support the full writing pipeline.
- Desktop-first experience: packaged with Electron for local installation, independent windows, and local data storage.

### What is included in this release

- Book and chapter management
- Autosave and editing support
- Model and feature configuration management
- Usage statistics and data management
- Chapter memory and settings memory workflows
- AI-powered drafting, expansion, summary, and review tools
- Initial GitHub Releases based auto-update support for future in-app upgrades

### Release Notes

- This is the **first official public release** of the project.
- The current release targets **Windows installers**.
- Auto-update infrastructure is now connected to GitHub Releases, preparing the app for smoother future upgrades.

### Who this is for

- Novelists and long-form fiction writers
- Creators who need structured character and worldbuilding support
- Users who want a local desktop AI writing environment instead of a browser-only workflow

Thank you for trying the first public version of AIWriter. Feedback and suggestions are very welcome.
