# 神笔写作 v0.9.4 | 首个公开可升级版本

## 中文介绍

欢迎来到 **神笔写作 v0.9.4**。

这是神笔写作在公开发布后的首个升级版本，也是项目第一次完整打通 **GitHub Releases + Electron 应用内自动更新** 的发布链路。你现在不仅可以下载安装包开始使用，也可以在后续版本中直接在应用内检查、下载并安装更新。

神笔写作是一款面向中文创作者的 AI 桌面写作工具，目标是把“作品管理、正文创作、设定维护、AI 辅助、数据本地化”整合到同一个稳定、可持续使用的写作工作台中。

### 产品定位

- 面向小说、网文、长篇故事、设定型内容创作
- 强调本地桌面工作流，而不是纯浏览器临时工具
- 强调结构化创作，而不是只提供一个简单对话框
- 强调长期写作支持，包括章节、设定、记忆、工具链和后续更新能力

### 核心能力

- 三栏式写作工作台：章节区、编辑区、AI 助手区同屏协作，减少切换成本。
- 本地作品管理：支持书籍、章节、全局设定、内容维护，适合持续创作。
- AI 辅助写作：支持正文创作、章节扩写、章节梗概、内容评估等高频任务。
- 设定与记忆体系：围绕角色、世界观、词条与向量检索构建持续上下文能力。
- 实用创作工具：内置脑洞生成器、书名生成器、简介生成器等工具，帮助从灵感走向成稿。
- 桌面端数据体验：本地安装、本地存储、独立窗口，更适合作为日常主力写作环境。

### v0.9.4 版本重点

- 正式将应用版本提升至 `0.9.4`
- 完成 GitHub Releases 发布流程整理
- 接入 Electron 应用内自动更新能力
- 移除旧的公告中心与自定义远端更新接口，统一更新入口
- 支持通过 GitHub Release 资产提供 `latest.yml`、安装包与 blockmap
- 完善首发后版本的发布说明与自动更新基础设施

### 这个版本适合谁

- 正在写小说、网文、长篇剧情内容的作者
- 需要持续维护角色与世界观设定的创作者
- 希望在本地桌面环境中使用 AI 辅助写作，而不是依赖浏览器网页工具的用户
- 想要一个后续可以持续升级的写作桌面应用的早期使用者

### 更新说明

如果你已经安装了旧版本，后续版本将可以通过应用内更新流程完成升级。对于首次下载的用户，这个版本也可以作为一个稳定的 Windows 安装入口。

感谢你继续关注神笔写作。欢迎持续反馈使用体验、创作场景和改进建议。

## English Introduction

Welcome to **AIWriter v0.9.4**.

This is the first upgrade release after the public launch of AIWriter, and it is also the first version that fully connects **GitHub Releases with Electron in-app auto update**. From this point on, users can not only download and install the app, but also receive future upgrades directly inside the desktop application.

AIWriter is a desktop AI writing application designed for Chinese-language creators. Its goal is to combine project management, drafting, structured settings, AI assistance, and local data ownership into one practical long-term writing workspace.

### Product Focus

- Designed for novels, serialized fiction, and long-form storytelling
- Built around a desktop workflow instead of a browser-only experience
- Focused on structured creation rather than a simple chat box
- Built for long-term writing support, including chapters, settings, memory, tools, and future updates

### Core Capabilities

- Three-panel writing workspace for chapters, editing, and AI collaboration
- Local project management for books, chapters, and global settings
- AI-assisted drafting, expansion, summaries, and review workflows
- Structured memory and worldbuilding support powered by vector-based context
- Built-in creativity tools such as idea generation, title generation, and synopsis generation
- Desktop-first local installation and storage for a more stable daily writing environment

### What is new in v0.9.4

- Application version updated to `0.9.4`
- GitHub Releases publishing workflow finalized
- Electron in-app auto update is now integrated
- Legacy announcement center and custom remote update endpoint removed
- Release assets now include `latest.yml`, installer package, and blockmap files
- Release notes and upgrade infrastructure improved for future versions

### Who this release is for

- Novelists and long-form fiction writers
- Creators managing structured characters and worldbuilding content
- Users who prefer a local desktop AI writing environment over web-only tools
- Early adopters looking for a desktop writing app that can continue to evolve through future upgrades

### Upgrade Note

If you already installed an older version, future versions can now be delivered through the in-app update flow. For first-time users, this version also serves as a stable Windows installation entry point.

Thank you for continuing to support AIWriter. Feedback, writing scenarios, and improvement suggestions are always welcome.
